# Clean NIYYAH Founder Hub

This is the cleaner, lighter founder hub. Use this instead of the big founder hub folder if GitHub upload is messy or keeps breaking.

## Folder To Upload

Upload everything inside:

```text
outputs/UPLOAD_THIS_CLEAN_FOUNDER_HUB_TO_GITHUB_2026_06_28/
```

This folder only contains:

```text
index.html
.nojekyll
docs/
```

## Recommended GitHub Repo

Create a repo called:

```text
niyyah-founder-hub
```

Then the stable link should be:

```text
https://niyyahhapp.github.io/niyyah-founder-hub/
```

## GitHub Steps

1. Open GitHub.
2. Create a new repo named `niyyah-founder-hub`.
3. Upload everything inside this folder.
4. Press `Commit changes`.
5. Go to repo `Settings`.
6. Go to `Pages`.
7. Choose `Deploy from a branch`.
8. Choose branch `main`.
9. Choose folder `/root`.
10. Save.

Wait 1-3 minutes, then open:

```text
https://niyyahhapp.github.io/niyyah-founder-hub/
```

## Why This Version Is Cleaner

The old founder hub copied the whole app, videos, waitlist, screenshots, and big Qur'an data file. That made uploads heavy and easier to break.

This clean version links out to the already-live app and waitlist instead.
